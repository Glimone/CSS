Elementos flutuantes

Nessa aula, iremos abranger os elementos flutuantes, utilizando 2 novos atributos:

  • Float
  • Clear

Float - Utilizado para fazer com que um containder flutue sobre o nível original do browser.

  • O conteúdo do browser irá agir como se o conteúdo em float não existisse, ficando por baixo.

  • Values: Left e right

Clear - Utilizado para definir se os elemento da página deve ficar ao lado dos elementos flutuantes.

  • Ao definir o clear, o conteúdo que iria ficar ao lado dos flutuantes, pode ser movido para 3 valores:

  • Right, left e both
