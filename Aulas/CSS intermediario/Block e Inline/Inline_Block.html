Aula de posicionamento com Block e Inline

Nessa aula, iremos ver sobre a definição de posicionamento natural dos elementos

Sendo as 3 definições: Block, Inline e Inline-Block.

Block

"Block" são elementos aplicados na vertical, sua tradução quer dizer "Bloco".

Exemplos:

<Div> | <H1> | <p>

Inline

"Inline" são elementos aplicados na vertical, sua tradução quer dizer "Bloco".

Exemplos:

<Span> | <a> | <b>

Porém, é possível tirar esse comportamento natural dos elementos, utilizando o atributo CSS Display.

O display possui como valores, as 3 definições citadas: Block, Inline e Inline-Block.

Display: Block

Torna os elementos do tipo inline em tipo block,

Esse texto é feito por um <Span>
Esse também, e estamos usando display:block; em ambos para mudar.
Display: Inline

Torna os elementos do tipo block em inline.

Nesse texto usamos <Div>
| Nesse usamos outra div, "|" é pra separar o texto, que está junto por ser inline, não se da espaço, só se optar.
Display: Inline-Block
Torna qualquer elementos em inline e quando não houver mais espaço, quebra a linha como block.
(Veja que é um novo paragráfo aqui) Quando não há mais espaço na linha, ele age como block.
OBS: Caso haja algum elemento dentro do elemento que está aplicando o display, o display não irá considerar esse elemento e o mesmo terá o comportamento natural.

Para que seja aplicado a mudança, também deve-se aplicar o mesmo display dentro desse elemento.
