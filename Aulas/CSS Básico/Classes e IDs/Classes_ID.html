Aprendendo sobre Class e IDs.

Class

O "class" deve ser estruturado dentro de "head", e adicionado na tag que deseja formatar.

Para criar um class, deve definir desse modo:

.class_name { atribute:definition; }

Depois, deve ser definido na tag, do seguinte modo:

<p class="Class_name"> (Conteúdo) </p>

ID

O "ID" é utilizado para um único fim na página, como por exemplo, um menu principal.

Por conta disso, um ID criado deve ser utilizado uma única vez no site.

Caso tenha uma outra coisa única, como um mmenu lateral e um superior, deve-se criar um ID para cada um deles, por serem únicos.

Definição de class:

#id_name { atribute: definition; }

Depois, deve ser definido na tag, do seguinte modo:

<h1 id="ID_name">(Conteúdo)</h1>

Essa é a explicação de class e ID.
