Aula de estilo de textos

  • Essa aula é destinada ao uso de estilos de textos, vistos anteriormente em HTML, porém, agora aplicados em CSS.

  Os estilos de textos que serão aplicados aqui são:

  • <b>, <strong> - (Bold; Negrito)
  • <em>, <i> - (Itálic; Itálico)
  • <u> - (Underline)
  • <Strike> - (Linha no meio dos caracteres; erro)

  Porém, aqui eles serão aplicados como atributos de CSS, então, essas tags não serão usadas. Elas devem ser usadas quando desejar individualizar uma palavra ou outra.


  Bold

  • Bold tem como função, deixar a palavra em negrito.

  • No CSS, deve ser aplicado dentro das chaves o atributo abaixo:
  • font-weight: bold;

  • Além do modo citado, também é possível definir o nível de negrito, basta colocar de 0 até 900.

  Italic

  • Itálic tem como função, deixar a palavra com itálico.

  • No CSS, deve ser aplicado dentro das chaves o atributo abaixo:
  • Font-Style: Italic;


  Underline

  • Underline tem como função, sublinhar uma palavra.

  • No CSS, deve ser aplicado dentro das chaves o atributo abaixo:
  • text-decoration: Underline;


  • Além do underline, que o prefixo Underline, quer dizer abaixo, temos o Overline, que quer dizer, acima. conforme o código já com a aplicação:

  • text-decoration: Overline;

  Strike

  • Strike tem como função, sublinhar uma palavra errada.

  • Ela é aplicada com o mesmo atributo do underline, entretanto, a definição é essa abaixo, que significa "Linha no meio".

  • No CSS, deve ser aplicado dentro das chaves o atributo abaixo:
  • text-decoration: line-through;


  • Essas são algumas das tags de CSS que aplicam fontes. Para ver outras diversas, clique no botão abaixo.
